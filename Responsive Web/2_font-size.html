font-size

직계 상위 요소에 font-size 속성이 없으면
16px = 100% = 1em = 1rem

  • font-size: 16px - 절대값
  • font-size: 100% - 상대값
    (부모요소의 font-size 속성에 대한 상대값)
  • font-size: 1em - 상대값
    (부모요소의 font-size 속성에 대한 상대값, em은 알파벳 M - 활자크기의 기준)
  • font-size: 1rem - 상대값
    (최상위 요소 html의 font-size 속성에 대한 상대값, rem은 Root Em(M)
