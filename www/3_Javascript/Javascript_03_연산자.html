연산자 - p.57

산술 연산자

  • (+)더하기
  • (-)빼기
  • (*)곱하기
  • (/)나누기
  • (%)나머지

대입 연산자

  • (=)대입하다
  • (+=)더하여 대입하다
  • (-=)제외하여 대입하다
  • (*=)곱하여 대입하다
  • (/=)나누어 대입하다
  • (%=)나누고 남은 나머지를 대입하다

증감 연산자

  • (++)1씩 증가
  • (--)1씩 감소

결합 연산자

  • (숫자+숫자+'문자')형식으로 입력
    - 연산자 좌우에 숫자가 있을 때 : 연산
    - 둘 중 하나 이상이 문자일 때 : (문자) 결합
  • 문자를 연결할 때 : (,)쉼표로 연결

비교 연산자

  • (>)크다
  • (<)작다
  • (==)같다
  • (!=)다르다, 같지 않다
  • (>=)크거나 같다
  • (<=)작거나 같다

논리 연산자(예시 생략)

  • and(&) 연산자 양 옆의 두 상황이 모두 true일 때 true의 결과값 제시- p.67
  • or(||) 연산자 양 옆의 두 상황중에서 하나 이상이 true일 때 true의 결과값 제시
  • not(!) 연산자 양 옆의 두 상황이 모두 false일 때 true의 결과값 제시

조건 연산자(예시 생략)

  • (조건) ? 'true값':'false값';
  • 조건을 만들고 결과값에 따라 실행값을 출력
