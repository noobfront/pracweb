동영상 연결

video 태그 사용

  • src (연결)
  • width (폭)
  • height (높이) : 미입력 시 width와 정비례
  • controls (조정판넬)
  • autoplay (자동실행)
  • muted (무음 : 자동실행 조건)
  • loop (반복)
  • playsinline (팝업 실행 차단)

iframe 태그 사용

  • iframe 태그는 다른 html 문서를 연결해서 보여주는 태그
  • 소스코드를 복사+붙여넣기

지도 연결

iframe 태그 사용 - google에서 주로 사용

  1. 구글 지도 검색
  2. 공유 버튼 클릭
  3. 지도 퍼가기 탭을 클릭하여 HTML 복사

스크립트 사용 - daum(권장)

로컬 실행할 때에는 보이지 않음

  1. 다음 지도 검색
  2. 위치 표시 아이콘 상단의 +(펼치기)
  3. 공유버튼 클릭 후 HTML 태그 복사 버튼 클릭
  4. 지도크기 지정 후 소스 생성
  5. 태그를 복사하여 붙여넣기
