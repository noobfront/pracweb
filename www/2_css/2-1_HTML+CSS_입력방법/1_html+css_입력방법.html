html+css 입력방법


css 형식

선택자{속성:속성값;속성:속성값}


선택자 유형

  1. tpye(태그) 선택자 - '태그명'을 그대로 입력
  2. class 선택자 - '.'+'클래스명'을 입력
  3. id 선택자 - '#'+'아이디명'을 입력
  4. 복합 선택자

선택자는 구체적일수록 우선순위가 높다.


css를 html에 적용시키는 방법

1. Inline Style Sheet

HTML 문서의 태그 안에 style속성을 직접 입력
- css입력내용이 매우 적을 때 사용
(우선순위 높음)

2. Internal Style Sheet

HTML문서의 head영역 안에 style태그를 삽입
- style태그에 type속성 입력
- 간단한 문서에 사용

3. Linking Style Sheet

별도의 css파일을 작성하여 html문서와 연결


- head영역 안에 link
