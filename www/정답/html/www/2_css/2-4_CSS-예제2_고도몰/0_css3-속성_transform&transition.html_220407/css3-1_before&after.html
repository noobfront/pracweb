가상클래스와 가상요소

가상클래스 선택자

  • 태그:nth-child(n) - 형제 요소 중 (n)번째
    (n)은 모든 요소, (숫자)번째
  • 태그:first-child - 형제 요소 중 맏이
  • 태그:last-child - 형제 요소 중 막내

가상요소 선택자 before&after

가상요소 선택자는 콜론 두 개(::)로 구분

before를 사용하여 경계 표시

before는 요소의 앞에 가상요소를 설정

  • 로그인
  • 회원가입
  • 마이페이지
  • 장바구니
  • 주문/배송

after를 사용하여 경계 표시

after는 요소의 뒤에 가상요소를 설정

  • 로그인
  • 회원가입
  • 마이페이지
  • 장바구니
  • 주문/배송
