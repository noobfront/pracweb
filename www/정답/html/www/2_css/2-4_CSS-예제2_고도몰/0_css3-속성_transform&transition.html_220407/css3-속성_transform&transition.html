transform은 변형 / transition은 과정

transform 속성

  • translate

    • 왼쪽 위를 기준으로 이동
    • transform: translate(X값,Y값) 형식으로 입력
    • transform: translateX() translateY() 둘로 나누어(띄어쓰기) 사용 가능
    • 마이너스(-)값 사용 가능

  • rotate

    • 시계방향으로 각도 회전
    • transform: rotate(수치deg) 형식으로 입력
    • deg(degree) 단위 사용
    • 마이너스(-)값 사용 가능

  • scale

    • 중심을 기준으로 확대/축소
    • transform: scale(배수); 혹은
      transform: scale(X배수,Y배수) 형식으로 입력
    • transform: scaleX() scaleY() 둘로 나누어 사용 가능
    • 기준은 1(=100%)

  • skew

    • 중심을 기준으로 기울어짐
    • transform: skew(X값,Y값) 형식으로 입력 - 값을 하나만 입력했을 때에는 X값으로 인식
    • transform: skewX() skewY() 둘로 나누어 사용 가능
    • deg(degree) 단위 사용
