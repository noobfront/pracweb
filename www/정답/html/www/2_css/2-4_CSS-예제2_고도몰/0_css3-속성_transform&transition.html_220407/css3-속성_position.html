css3 속성

position 속성 - P.362

  • static

    고정 위치 (기본설정)

    • 기본설정이므로 별도로 설정하지 않음

  • relative

    상대적 위치

    • left: top: right: bottom: 등 위치값 사용
    • 위치값의 기준은 현재 위치
    • Z값의 우선순위 높음

    하위 absolute

    직계상위(조상)요소 중 position 속성(기본속성인 static 제외)을 가진 요소가 있으면 그 요소의 위치를 기준으로 삼음

  • absolute

    절대적 위치

    • left: top: right: bottom: 등 위치값 사용
    • 위치값의 기준은 브라우저의 body태그 왼쪽 위
    • 영역을 차지하지 않음
    • Z값의 우선순위 높음

    하위 absolute

    직계상위(조상)요소 중 position 속성(기본속성인 static 제외)을 가진 요소가 있으면 그 요소의 위치를 기준으로 삼음

  • fixed

    절대적 고정 위치

    • left: top: right: bottom: 등 위치값 사용
    • 위치값의 기준은 브라우저의 body태그 왼쪽 위
    • 영역을 차지하지 않음
    • Z값의 우선순위 높음
    • 스크롤 고정

    하위 absolute

    직계상위(조상)요소 중 position 속성(기본속성인 static 제외)을 가진 요소가 있으면 그 요소의 위치를 기준으로 삼음
